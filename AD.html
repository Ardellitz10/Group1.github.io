Installing AD 

 

1.  In Server Manager, go to “Manage” and select “Add Roles and Features”  

 
 
 
 

 
 
 
 
 

2.  In “Installation Type”, select “Role-based or feature-based installation” and click 

next 

 

3.  In “Server Roles”, click the box next to “Active Directory Domain Services” 

-  Click “Add Features” afterwards  

 

4.  Keep clicking next until the option to Install is available, then click “Install” 

 

5.  Once finished installing, select “Promote this server to a domain controller” 

 

 

6.  Under “Deployment Configuration”, select “Add a new forest” 
7.  Enter your root domain name 

-  Click next  

 

8.  Under “Domain Controller Options”, do the following:  

-  Make sure “Domain Name System (DNS) server” is selected  
-  Make sure “Global Catalog (GC)” is selected 
-  Enter a password for the DSRM 
-  Click next  

 

9.  Under “Additional Options” enter your NetBIOS domain name, then click next 

 

10. Make sure the location of each folder is going to their respective location, click next 

afterwards 

 

11. Review all your selections to make sure they are correct, then click next 
12. Under “Prerequisites Check”, Click install 
13. Sign in after the VM restarts
