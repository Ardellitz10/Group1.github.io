Group Windows 11 Install Documentation

(Jonathan Puddie, Benjamin Gustafson, Joshua Ardellitz)

 

Creating a virtual machine:

 

1.    Go to ESXi server 10.25.1.3

-        Note: Make sure youre connected to a VPN to use VMware ESXi

2.     Sign in with your SVCCISDEPT account

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


3.       On the left-hand side where it says virtual machines, right click and select create/register VM

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


4.       Hit Create a new virtual machine and click next

 

 

 

 

 

 

 

 

 

 

 


5.       Name the VM WIN11Group#

6.       Change the guest OS family to windows

7.       Change guest OS version to Windows 10 (64-bit), then click next

 


 

 

 

 

 

 

 

 

 

 


8.       Select DataStore1 and click next

 

 

 

 

 

 

 

 

 

 


9.       Assign 4 CPUs

10.  Change the RAM to 16 GB (under Memory)

11.  Change hard drive to 80 GB (under Hard disk 1)

12.  Under Network Adapter 1, change it to VM network

 

 

 

 

 

 

 

 

 

 

 

 


13.  Under CD/DVD drive 1 attach iso file (click dropdown) select data store iso file click DataStore1 ISO FILES Windows 11 then click next

14.  Click finish

 

 

Installing Windows 11:

 

 

 

 

 

 

 

15.  Start the VM

16.  Upon the first boot make sure to get into the VM quick and hit any key to boot from CD

 

 

 

 

 

 


17.  Select preferred language and click next

 

 

 

 

 

 

 

 

 


18.  Click Install Now

 

 

 

 

 

 

 

 

 

 


19.  Before you click which version to install, we have to modify the registry

20.  Press shift + F10 to pull up the command prompt and type regedit

21.  Next go to HKEY_LOCAL_MACHINE SYSTEM right click on setup then new and key

 

22.  Name the key LabConfig spelled this way with the capitals and no spaces

 

 

 

 

 

 

 

 

 

 


23.  Right click in the empty white space and select new dword 32 bit

 

24.  Name this ByPassTPMCheck, again spelled this way

 

 

 

 

 

 

 

 

 


25.  Double click the name and change the value to 1

26.  Exit out of that and the command prompt

 

 

 

 

 

 

 

 

 

 

 

 


27.  Select Windows 11 Pro Education and hit next

 

 

 

 

 

 

 

 

 

 

 


28.  Click the I accept box and then click next

29.  Click custom install

 

 

 

 

 

 

 

 

 

 

 


30.  Select your 80 GBhard drive and hit next

31.  Wait for it to install (takes about 10-15 minutes)

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


32.  Click your country/region and click Yes (Make sure it is selected as United States)

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


33.  Select input method and click yes and hit skip when it asks to add a second keyboard layout

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


34.  Name your device Win11Group# (appropriate group number for your group)

 

 

 

 

 

 

 

 

 

 

 

 

 

 


35.  Select set up for work or school click next

 

 

 

 

 

36.  Click the sign in options below where it says to sign in

37.  Select the domain option

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 

 


38.  Set the name to Group#user and click next

 

 

 

 

 

 

 

 

 

 

 

 


39.  Make the password, Rajah424!, and click next

40.  Confirm the password and then click next

 

 

 

 

 

 

 

 

 

 

 

 

 


41.  Answer a security question and click next do this for all three questions

 

 

 

 

 

 

 

 

 

 

 


42.  Click Next and wait for it to load (might take a few minutes)

 

Licensing the machine:

 

 

 

 

 

 

 

 

 

 

 


43.  Next, license the machine by going to settings system activation and edit the product key and type in GY3NJ-CGTT9-MJKBK-VQRP3-4GFBQ

44.  Shut down the VM

45.  Now install vmware tools

 

 

 

 

46.  Go back into vmware and under your VM screen will be vmware tools

47.  Go to actions guest OS install vmware tools

48.  Boot the VM back up and log in

 

 

 

49.  Go to file explorer and go down to DVD drive (D:) VMware tools

50.  Click on that and hit setup then click yes to install

51.  Once done click finish and restart
