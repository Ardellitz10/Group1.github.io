Creating the Linux box 

1.  Log onto VMware 
2.  Go to and select “Create / Register VM” 
-  Select “Create a new virtual machine” 

 

3.  Enter for the following:  
-  Name: Group#_LinuxBox 
-  Guest OS family: Linux  
-  Guest OS version: Debian GNU/Linux 11 (64-bit) 
4.  In “Select storage”, select “DataStore1” 

 

5.  Enter the following: 
-  Network Adapter 1: Group# 
-  CD/DVD Drive 1: Datastore ISO file  

•  When selecting the ISO file: 

 

Go to “ISO FILES” → “MINT” → most recent version of Linux mint shown here 

•  Click “Select” 

6.  Click “Next” and then click “Finish”  

 

7.  Right click your new VM, hover over “Power”, and select “Power on” 

 

 

 

8.  When powered on, select the first option by pressing “enter” 

 

 

9.  At the top left corner of the screen, click on “Install Linux Mint” 
10. Select “English” as your language and “English (US) as your keyboard layout 

 
 
 

 

11. Check the box for “Install multimedia codecs” 
-  Click continue afterwards 

 

12. Select “Erase disk and install Linux Mint” 
-  Click “Install Now” afterwards 
-  Select “Continue” under “Write the changes to disks?” 

 

 

13. Select “New York” under “Where are you?” 

 

14. Enter the following: 
-  Your name: Linux# 
-  Password: Rajah424! 
-  Select continue when finished 

NOTE: “Pick a username” will be automatically filled when you enter your name 

 

13. Click “Restart Now” when the installation is complete 

 

14. Log onto your Linux VM 

 

15. In the bottom right corner, left click the network icon and select “Network Settings” 

 

 

 

 

16. In the bottom right corner, click the gear icon 

 

17. Under “IPv4”, change “Addresses” to “Manual” and change the following settings: 
-  Address: 172.16.1.55 
-  Netmask: 255.255.255.0 
-  Gateway: 172.6.1.1 

 

18. Additionally, change the following settings: 
-  Server: 172.16.1.1 

Click the “+” icon to add a server and add 8.8.8.8 

-  Click Apply once finished 

 

19. Restart Linux Mint and log in to confirm you’re connected
