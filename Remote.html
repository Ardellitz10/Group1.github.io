Remote Access on Windows 11 

Group 1 (Josh Ardellitz, Benjamin Gustafson, Jonathan Puddie) 

 

 

 

 

1.  On the Windows 11 VM open the terminal/command prompt 
2.  Enter ipconfig 
3.  Find the IP associated with IPv4 Address (in this case 172.16.1.23) 
4.  Write this Address down for later  

 

5.  In the search enter Control Panel 

 

6.  Select system and security  

 

 

7.  Under System Select Allow Remote Access  
8.  Enter an admin username and password and select continue 

9.  Under Remote Desktop Select the bubble that says Allow remote connections to 

this computer 

10. Select apply 
11. Select select users  

 

12. Select Add 

13. Select Advanced  
14. Select Find now 

15. Select the users and groups you want to have access to the remote connection  

 

16. Head over to your group server VM 

17. In the search bar enter remote desktop connection and open it 

18. Enter the IP address that you wrote down from earlier  

19. Select connect 

20. Should prompt you to sign in to the account then you will have remote access to 

that desktop.
