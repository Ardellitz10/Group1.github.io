IIS Setup 
1. In the Windows Server VM, open “Server Manager” 

 

2. Select “Add roles and features”  

3. Click “Next” until you reach “Server Roles” 

 

4. Under “Server Roles”, select “Web Server (IIS)”, and select “Add Features”  

 

 

5. Keep clicking “Next” until you reach “Confirmation”  

- Select “Install”  

 

6. Once done installing, go to a browser and search “127.0.0.1” to verify IIS is working  

 

 

Installing Apache 

 

1.  In Linux, open “Terminal” 

 

2.  Enter “sudo apt install apache2”, and press enter once finished 

-  Enter your password when prompted  
-  Enter “Y” when prompted to continue 

3.  Once done installing, search “127.0.0.1” in a web browser to verify Apache2 

installed
