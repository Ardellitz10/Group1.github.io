Group 1 Disabling default admin account & adding new admin 

(Benjamin Gustafson, Joshua Ardellitz, Jonathan Puddie) 

Adding new admin account: 

1.  On the server VM, open “Active Directory Users and Computers” 
2.  Click on the folder called “Users” and select “New” → “User” 

3.  Name your new administrator account 

4.  Create a password (In this case CS265CS265!) and select “User cannot change 

password” 

5.  Select “Next” and then click “Finish” 

6.  Under the new user right click “All Tasks” and select “Add to a group” 

7.  Enter “Administrators” and select “Check Names” 

-  Click “OK” afterwards  

Disabling Default Admin 

1.  Right click on the new administrator account 

-  Select “All Tasks” and then select “Disable Account”
