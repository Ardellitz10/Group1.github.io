Group 1: Adding a new user with a home folder  

(Benjamin Gustafson, Joshua Ardellitz, Jonathan Puddie) 

 

1.  At the bottom of your screen, search “Active Directory User and Computers” 

-  Click “Open”  

 

2.  Under your domain name, click the drop-down arrow 

-  Right click “Users” 

 

3.  Hover over “New” and select “User” 

 

4.  Create your user  

-  Click Next 

 

5.  Create your password  

•  NOTE: Make sure “User cannot change password” is checked 
-  Click “Next” 

6.  Click “Finish”  

 

 

 

 

 

 

 

 

 

 

 

7.  Open “File Explorer”, select “This PC”, select “Local Disk (C:)” 

8.  In “Local Disk (C:)”, right click and select “New”, and select “Folder”, name the 

folder “Home” 

9.  Right click the “Home” folder → “Properties” → “Sharing” → “Advanced Sharing” → 

“Permissions” 

 

10. Remove “Everyone” under “Group or user names:” 
11. Click “Add” and enter “Domain Users”  

-  Click “Check Names” 
-  Make sure “Full Control” is checked 
-  Click “Ok” once you’re done 

12.  In “Active Directory Users and Computers”, right click the user 

-  Click “Properties” 

 

 

 

 

 

 

 

 

13. Click “Profile” 
14. Click “Connect” under “Home folder” 
15. Select the H: \ drive 
16. Under “To:”, Enter \\ server ip address \ Home \ Martin McGuinness 

-  Click “Apply” and “OK” once finished 

17. In “File Explorer”, right click “This PC” and select “Map network drive” 

 

 

18. Select H under “Drive” 
19. Enter your folder location and select “Finish”
