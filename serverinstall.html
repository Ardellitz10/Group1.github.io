Group 1 Server Install  

(Benjamin Gustafson, Joshua Ardellitz, Jonathan Puddie) 

Setting up the server: 

 

1.  Go to 10.25.1.3 to log onto VMware EXSi 

 

 

 

 

 

 

 

 

 

2.  Once logged in, go to “Virtual Machines” and click “Create / Register VM” 
3.  Under “Select creation type”, select “Create a new virtual machine” and click next 

 

4.  Under “Select a name and guest OS”: 

-  Name the VM “Group#_Server” (Name it what group number you are)  
-  Select “Windows” under “Guest OS family” 
-  Select “Microsoft Windows Server 2022 (64-bit)” under “Guest OS Version” 
-  Hit next once finished 

 

5.  Under “Select storage”, select “DataStore1”, then click next  

 

 

 

 

 

 

 

 

 

 

6.  Under “Customize settings”: 

-  Set the CPU to 4 
-  Set the memory to 4 GB 
-  Set “Hard disk 1” to 80 GB 
-  Set “Network Adapter 1” to your group name 
-  Set “CD/DVD Drive 1” to “Datastore ISO file” 

 

 

 

 

 

 

 

 

 

▪

 

Open “DataStore1” 

▪

 

Open “ISO FILES” →Windows Server → Select the most recent windows 
server 

▪

 

NOTE: Click the drop-down arrow to make sure that the box “Connect at 
power on” is checked, also make sure the box to the right of CD/DVD Drive 1 
named “Connect” is also checked 

-  Click next 

7.  Click finish  

 

8.  Under “Actions”: 

-  Hover over “Guest OS” → Install VMware Tools 
 

Using the VM: 

9.  Power on the VM 

NOTE: Upon the first boot make sure to get into the VM quick and hit any key to boot 
from CD 

 

 

10. In “Boot Manager”, select “EFI VMware Virtual SATA CDROM Drive (0,0)” and press 

enter 
NOTE: You will have to use the arrow keys to navigate between the options 

11.  Select English as your language  
12. Select “US” as your keyboard or input method 
13. Under “Select setup option”, select “Install Windows Server” and click the check 

box to agree 
 
 
 
 
 
 
 

 

 

 

 

 

 

 

 

 

 

 

14. Under “Choose a licensing method”, enter the product key provided to you 

-  Click next 

 

 

 

 

 

 

 

 

 

 

 

 

15.  Select “Windows Server 2025 Standard (Desktop Experience)”  

-  Click next 

16. Under “Applicable notices and license terms”, click accept 

 

17. Click next  
18. Under “Ready to install”, click install 
19.  Allow Windows Server to be installed (Estimated time: 15-20 minutes)  

 

20. Create a password to use to log in  

21. Log into your administrator account  

 

-  NOTE: To press “ctrl + alt + delete” within the VM: 

▪

 

Go to “Actions” at the top right corner of the VM tab 

▪

 

Select Guest OS → Send keys →Ctrl-Alt-Delete 
 

23. Go to settings → Check for updates 
24. Install any updates
